Rock'em Sock'em

This is a version of the 1964 Rock’em Sock’em Robots game that was built with Materialize, jQuery and Javascript.

starsGameplaystars

  1. Select a player.
  2. Select a battleground.
  3. Punch or kick your opponent.
  4. Celebrate victory!

Keyboard keys/symbols

Player 1 Player 2
Q = punch P = punch
A = kick L = kick
Z = run left left arrow = run left
X = run right right arrow = run right

There are additional moves that can be used during the game by clicking on the buttons for each action. There is also a damage meter to track how much life is left. When a player's healt meter is at zero, the opposing player wins the game.

Features

The game displays motivational quotes and commentary from the Chuck Norris API.

The Internet Chuck Norris Database @ http://www.icndb.com
